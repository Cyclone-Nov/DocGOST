Как предполагается использовать ПО?
1. Отдел конструирования:
   Сейчас: так как программа по созданию спецификаций перестала работать, то создание спецификаций на основе шаблона из Excel вручную и преобразование в pdf. Спецификация, объединяющая детали и электронные компоненты, создается с помощью программы.

   Будет: на овнове выгружаемого из Solid Works файла ПО открывает спецификацию, позволяет редактировать ее и экспортировать в pdf или excel. В идале экспорт в excel не нужен - он может пригодится только для если в ПО не удалось отредактивровать спецификацию как надо.

3. Отдел документации:
   Сейчас: выбираются из PDM файлы спецификаций и далее объединяются данные из них по компонентам вручную в Excel, считая кол-во компонентов вручную.
   Будет: ПО автоматически будет объединять файлы спецификаций для составления ведомости покупных изделий. Из конструкторского отдела будут указывать какой из файлов является главным и на основе его ПО будет подгружать остальные файлы спецификаций для сборочных единиц для построения ведомости компонентов
